# UROP---project-Detect-hallucinations-in-LLMs-
Detecting and Explaining Hallucinations in Large Language Models

Large language models sometimes produce confident-sounding but factually incorrect statements; a phenomenon known as hallucination. In RAG systems, an LLM generates answers based on retrieved source documents, but may still produce claims not grounded in any source. The main objective is to build a tool that compares each claim in an AI-generated response to the retrieved documents and classifies it as supported, partially supported, or unsupported.
